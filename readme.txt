== Description ==

Smart Coming Soon Mode is a simple and lite plugin that allows you to create a beautiful and clean Smart Coming Soon Mode Page / Under Maintenance Page / Under Constructions Page / Pre-Launch Page for your website and share your contact information with users and connects with users on your social networks.

The Smart Coming Soon Mode works with any WordPress theme you have installed on your website.


**Features**

* Fully Responsive Design
* Enable/Disable Smart Coming soon mode
* Enable/Disable Site logo
* Enable/Disable Countdown Timer
* Set End Date Countdown Timer
* Heading Line Text
* Description Text Message
* Color and Image Background supported
* Show/Hide Overlay/Dim Background  
* All Text Color Settings
* Enable/Disable Social Icons Settings
* Social Icons (Facebook, Twitter, LinkedIn, Google) Supported
* Enable/Disable Contact Information
* Contact Info Settings
* Typography
* Live Preview Page
* Works with any WordPress theme you have installed on your website
* Easily To Customize Setting Option Panel
* Single Default Template Desing
* Any More Features


== Installation ==
In WordPress:

1. Go to `Plugins` > `Add New` > search for `Smart Coming Soon Mode`
2. Press `Install Now` for the Smart Coming Soon Mode plugin
3. Press `Activate Plugin`

Manual installation:

1.Download Smart Coming Soon Mode plugin.
2. Upload the entire **smart-coming-soon-mode** folder to the **/wp-content/plugins/** directory.
3. Activate the plugin through the **Plugins** menu in WordPress admin.
